# 介绍：

原始代码无加密，棋牌游戏源码，斗地主、麻将、跑得快、德州扑克、捕鱼等。

# 源码包含：

1. 系统模块（服务端、pc客户端、控制端）源代码；
2. LUA移动端（ios、android）源代码；
3. N款子游戏源代码（包含游戏看上方截图）；
4. 新3.0内核源代码；
5. 大厅ui工程源代码（方便移动端换皮）；6.前端网站及后台源代码；
6. 数据库自建脚本；
7. 游戏配置工具及成品组件
